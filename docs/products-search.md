# Поиск товаров

Данный эндпоинт позволяет выполнять поиск товаров по множеству фильтров: категориям, цене, весу, размерам, производителям, продавцам, рейтингу и характеристикам (вариантам). Поддерживает пагинацию и возвращает детальную информацию для построения фильтров на фронтенде (минимальные/максимальные значения, доступные варианты, списки производителей и продавцов).

## Информация о запросе

| Свойство  | Значение               |
| --------- | ---------------------- |
| Метод     | `GET`                  |
| URL       | `/api/products/search` |
| Тип формы | `query`                |
| Имя URL   | `products_search`      |

## Параметры запроса (Query Parameters)

Запрос передается в формате JSON в теле `GET`-запроса.

### Основные фильтры

| Поле       | Тип      | Описание                                                                                      |
| ---------- | -------- | --------------------------------------------------------------------------------------------- |
| `filters`  | `object` | Объект с условиями фильтрации (см. таблицу ниже).                                             |
| `paginate` | `object` | Объект с настройками пагинации: `page` (номер страницы) и `pp_items` (элементов на странице). |

### Структура объекта `filters`

Объект содержит множество полей для фильтрации. Наиболее важные из них:

| Поле                                  | Тип      | Описание                                                 |
| ------------------------------------- | -------- | -------------------------------------------------------- |
| `categories`                          | `array`  | IDs категорий для фильтрации.                            |
| `search_string`                       | `string` | Поисковый запрос по названию/описанию.                   |
| `price_from` / `price_to`             | `string` | Диапазон цен.                                            |
| `product_manufacture`                 | `array`  | IDs производителей.                                      |
| `product_owner`                       | `array`  | IDs продавцов.                                           |
| `avg_raiting_from` / `avg_raiting_to` | `string` | Диапазон среднего рейтинга.                              |
| `with_discount`                       | `string` | Только товары со скидкой (`true`/`false`).               |
| `weight_from` / `weight_to`           | `string` | Диапазон веса.                                           |
| `length_from` / `length_to`           | `string` | Диапазон длины.                                          |
| `width_from` / `width_to`             | `string` | Диапазон ширины.                                         |
| `height_from` / `height_to`           | `string` | Диапазон высоты.                                         |
| `ordering`                            | `string` | Сортировка (например, `price`, `-price`, `avg_raiting`). |
| `variant_1`, `variant_3` и др.        | `array`  | Фильтры по характеристикам (цвет, размер и т.д.).        |

### Пример запроса

```json
{
  "filters": {
    "categories": [1, 3],
    "price_from": "500",
    "price_to": "5000",
    "with_discount": "true",
    "ordering": "-avg_raiting"
  },
  "paginate": {
    "page": 1,
    "pp_items": 15
  }
}
```

## Информация об ответе

### Основные поля ответа

| Поле              | Тип                 | Описание                                                         |
| ----------------- | ------------------- | ---------------------------------------------------------------- |
| `form_errors`     | `null` или `object` | Ошибки валидации формы. При успешном запросе — `null`.           |
| `success_message` | `string`            | Сообщение об успешном выполнении.                                |
| `page_data`       | `object`            | Основной объект с данными ответа (см. детальную структуру ниже). |
| `spent`           | `float`             | Время выполнения запроса на сервере в секундах.                  |
| `locale`          | `string`            | Код языка, использованный для локализации ответа.                |
| `url_name`        | `string`            | Техническое имя эндпоинта.                                       |

### Структура объекта `page_data`

| Поле               | Тип       | Описание                                                        |
| ------------------ | --------- | --------------------------------------------------------------- |
| `min_max`          | `object`  | Минимальные и максимальные значения цен, веса, размеров.        |
| `variants`         | `object`  | Доступные варианты товаров (цвет, размер и т.д.).               |
| `selects`          | `object`  | Списки производителей, продавцов, категорий.                    |
| `measures`         | `object`  | Детальные данные по измерениям.                                 |
| `discounts`        | `object`  | Информация о скидках (например, количество товаров со скидкой). |
| `raiting_above_45` | `integer` | Количество товаров с рейтингом выше 4.5.                        |

#### Детальная структура `variants`

Объект `variants` содержит характеристики товаров. Ключи — ID характеристик (например, `"variant_1"` для цвета). Каждая характеристика содержит:

| Поле      | Тип       | Описание                                                                        |
| --------- | --------- | ------------------------------------------------------------------------------- |
| `id`      | `integer` | ID характеристики.                                                              |
| `locale`  | `object`  | Локализованное название характеристики (`variant_type_name`) для разных языков. |
| `options` | `array`   | Массив доступных значений для этой характеристики.                              |

Каждый элемент массива `options` содержит:

| Поле     | Тип       | Описание                                            |
| -------- | --------- | --------------------------------------------------- |
| `id`     | `integer` | ID значения (опции).                                |
| `locale` | `object`  | Локализованное название значения для разных языков. |
| `color`  | `string`  | HEX-код цвета (если характеристика — цвет).         |
| `count`  | `integer` | Количество товаров с этим значением.                |

#### Структура `selects`

| Поле          | Тип     | Описание                                                                                          |
| ------------- | ------- | ------------------------------------------------------------------------------------------------- |
| `manufacture` | `array` | Массив производителей. Каждый объект содержит `id`, `name`, `image`, `count` и др.                |
| `sellers`     | `array` | Массив продавцов. Каждый объект содержит `id`, `seller_name`, `seller_raiting`, `count` и др.     |
| `categories`  | `array` | Массив категорий. Каждый объект содержит `id`, локализованное название (`cg_name`), `count` и др. |
