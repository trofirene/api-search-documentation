## Информация о запросе

| Свойство         | Значение             |
| :--------------- | :------------------- |
| **Метод**        | `POST`               |
| **URL**          | `/api/addresses/get` |
| **Тип контента** | `application/json`   |
| **Имя URL**      | `address_get`        |

Данный эндпоинт позволяет получить справочную информацию о странах и адресах пользователя. Возвращает список стран с их кодами и локализованными названиями, а также список сохраненных адресов текущего пользователя.

---

## Параметры запроса (Request Body)

### Основные поля

| Поле       | Тип      | Обязат. | Описание                      |
| :--------- | :------- | :-----: | :---------------------------- |
| `filters`  | `object` |   ❌    | Объект с условиями фильтрации |
| `paginate` | `object` |   ✅    | **Настройки пагинации**       |

### Детальная структура

#### Объект `filters`

| Поле  | Тип      | Обязат. | Описание                                                                      | Пример |
| :---- | :------- | :-----: | :---------------------------------------------------------------------------- | :----- |
| _нет_ | `object` |   ❌    | В текущей версии фильтрация не поддерживается. Передавайте пустой объект `{}` | `{}`   |

> 💡 **Для будущего использования:** Структура предусматривает возможность добавления фильтров. Следите за обновлениями документации.

#### Объект `paginate`

| Поле       | Тип       | Обязат. | Описание                         | Пример |
| :--------- | :-------- | :-----: | :------------------------------- | :----- |
| `page`     | `integer` |   ✅    | Номер страницы (начиная с 1)     | `1`    |
| `pp_items` | `integer` |   ✅    | Количество элементов на странице | `10`   |

---

### Пример запроса

```json
{
  "filters": {},
  "paginate": {
    "page": 1,
    "pp_items": 10
  }
}
```
